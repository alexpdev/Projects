# Bitprint

A tiny CLI for converting integers to binary.

## Dependencies

- Python 3.*

## Installation

```powershell
git clone https://github.com/alexpdev/bitprint.git
cd bitprint
python setup.py install
```

## Usage

```powershell
> bitprint 1
1
> bitprint 2
10
> bitprint 12
1100
```
